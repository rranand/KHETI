Geek Information

All the technical particularities used...

Vegetation Index

With the help of drone imaging, the application we make the NDVI from those images and then would segement those images into blocks where the vegetation index is quite low.

Weed detection

After scanning the whole farm, all the images captured would be uploaded on this platfrom and the platform will scan the images with the deep learning Xception model and will give the GPS locations and also the shortest path to reach that point with the help of google maps API.

Disease Prediciton

The farmer can take the image of the leaf of the plant that he suspects to have some or other kind of disease and the system with the help of deep learning ResNet Model will give prediction of which disease the crop is having.

Fertilizer Recommendation

With the help of N-P-K, pH, soil and farm crop information the system will give information on which fertilizer to use with the help of RandomForest machine learning algorithms.

Crop Recommendation

With the help of N-P-K, pH, soil and rainfall information the system will give information on which crop to grow with the help of MLP machine learning algorithms.
